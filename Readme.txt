Git command

1. git init
2. git status
3. git add . (file/folder)
4. git commit -m "first commit"
5. git log -oneline
6. git checkout 90073cc index.html   // go back to previous file
7. git reset HEAD index.html // to show git status in red color
8. git checkout -- index.html // come back to last commit
